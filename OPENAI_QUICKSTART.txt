📝 STEP 1: GET YOUR API KEY
────────────────────────────────────────────────────────────────────────────────
1. Visit: https://platform.openai.com/api-keys
2. Click "Create new secret key"
3. Copy the key (starts with sk-proj- or sk-)
4. Keep it secret!


🔧 STEP 2: ADD TO PROJECT
────────────────────────────────────────────────────────────────────────────────
Edit .env file:
    nano .env

Replace this line:
    OPENAI_API_KEY=your_openai_api_key_here

With your actual key:
    OPENAI_API_KEY=sk-proj-xxxxxxxxxxxxxxxxxxxxxxxxxxxxx


🚀 STEP 3: RESTART BACKEND
────────────────────────────────────────────────────────────────────────────────
    pkill -f "python3 app.py"
    python3 app.py

Look for this message:
    ✓ OpenAI API enabled    <-- Good!

Instead of:
    ℹ OpenAI API key not configured - using rule-based NLP


✅ STEP 4: TEST IT
────────────────────────────────────────────────────────────────────────────────
Run the test suite:
    python3 test_openai.py

Or test manually in browser:
    http://localhost:3000

Try these queries:
    • "What's the average transaction for food on Android?"
    • "Compare weekend vs weekday performance"
    • "Show me peak hours for entertainment category"


💡 WHY USE GPT?
────────────────────────────────────────────────────────────────────────────────
✓ Understands natural language variations
✓ Handles typos and colloquialisms  
✓ Maintains conversation context
✓ Extracts complex filters automatically
✓ Works with ambiguous queries

WITHOUT GPT (rule-based):
    Query: "how much people spend on food?"
    Result: ❌ May not understand

WITH GPT:
    Query: "how much people spend on food?"
    Result: ✓ Understands → average → Food category → Perfect answer!

🎯 CUSTOMIZATION OPTIONS
────────────────────────────────────────────────────────────────────────────────
In app.py, you can customize:

1. MODEL SELECTION:
   Current: model="gpt-3.5-turbo"
   Upgrade: model="gpt-4o-mini"  (smarter, 2x cost)

2. TEMPERATURE:
   Current: temperature=0.2  (focused)
   Creative: temperature=0.7 (more varied)

3. MAX TOKENS:
   Current: max_tokens=400
   Increase: max_tokens=800  (for longer responses)

4. SYSTEM PROMPT:
   Customize for your specific business needs
   Add more context about UPI transactions
   Include industry-specific terminology

⚠️ TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────
ISSUE: "OpenAI initialization failed"
FIX: Check API key format in .env

ISSUE: "Rate limit exceeded"
FIX: Upgrade your OpenAI tier or add delays

ISSUE: Still using rule-based NLP
FIX: Restart backend after editing .env

ISSUE: High costs
FIX: Use GPT-3.5-turbo (not GPT-4)
     Reduce max_tokens
     Set spending limits in OpenAI dashboard
